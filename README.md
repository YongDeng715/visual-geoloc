# visual-geolocalization
source code of Visual Geo-localization project, which aims to match images of UAV and Satellites in GPS-denied condition.


## Acknowledgement
This project is based on the following papers:

1. [Visual place recognition for aerial imagery: A survey](https://www.sciencedirect.com/science/article/abs/pii/S0921889024002215), [aero-vloc](https://github.com/prime-slam/aero-vloc)
2. [UAV-VisLoc](https://github.com/IntelliSensing/UAV-VisLoc)
3. [UAV-GeoLoc](https://github.com/RingoWRW/GeoLoc-UAV)
4. [SelaVPRplusplus](https://github.com/Lu-Feng/SelaVPRplusplus)
